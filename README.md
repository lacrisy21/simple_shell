## Deployment ⚙️

#### Requirements
 - Gcc >= 4.8.4

#### Steps to compile the shell and execute.
 - `git clone https://github.com/lacrisy21/simple_shell.git`
 - `cd simple_shell`
 - `gcc -Wall -Werror -Wextra -pedantic *.c -o thanatos`
 - `./thanatos`
